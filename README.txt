[bd] Paygates

Open Source: https://github.com/xfrocks/bdPaygates

[FEATURES]
1. Replace built-in PayPal with compatible implementation (no change needed to your system)
2. Support other pay gates as additional download.
3. Support paid resource (in XenForo Resource Manager)

[INSTALLATION]
1. Upload the library directory into your XenForo's
2. Import the add-on
3. Configure pay gate options in User Upgrade page (AdminCP)

[ALSO AVAILABLE ADD-ON]
- Authorize.Net: http://xenforo.com/community/resources/1605/
- BitPay.Com: http://xenforo.com/community/resources/1656/
- JahanPay.com: http://xenforo.com/community/resources/2523/
